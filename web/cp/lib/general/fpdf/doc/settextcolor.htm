SetTextColor

SetTextColor(int r [, int g, int b])

Version

1.3

Descripción

Define el color usado por el texto. Este puede ser expresado en componentes RGB o escala de grises. El método puede ser invocado antes que la primera página sea creada y el valor es retenido de página a página.

Parámetros

r
Si g y b son suministrados, componente rojo; si no, indica el nivel de grises. Valor entre 0 y 255.
g
Componente verde (entre 0 y 255).
b
Componente azul (entre 0 y 255).

Vea además

SetDrawColor(), SetFillColor(), Text(), Cell(), MultiCell().
